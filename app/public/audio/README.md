# Herd & Seek audio

The WAV files in this directory are original, deterministic compositions and sound designs generated by `scripts/generate-audio-assets.mjs`. They contain no third-party recordings, samples, melodies, or dependencies and are licensed with the Herd & Seek project for unrestricted use in the game and its deployments.

Run `npm run assets:audio` to regenerate every file byte-for-byte.
